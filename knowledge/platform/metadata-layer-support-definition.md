# Metadata Layer Support Definition

## Summary

At the metadata layer, platform support means only that a release exposes records relevant to a platform role. It does not mean a device works, a driver starts, or a target can boot.

## Definition

Metadata-layer support: the presence of reduced, provenance-backed metadata that can identify a platform-facing role, matching surface, or candidate relationship.

## Non-Claims

- hardware enablement
- runtime attachment
- driver behavior
- boot viability
- compatibility
