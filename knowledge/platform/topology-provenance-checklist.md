# Topology Provenance Checklist

## Summary

A reduced topology record is reproducible only when it names the plane, command or artifact source, redaction policy, and schema shape. Topology without provenance should not feed release comparison.

- Related schema: `datasets/schemas/ioreg-inventory.schema.json`

## Checklist

- [ ] Plane name.
- [ ] Source command or artifact.
- [ ] Host/release provenance.
- [ ] Reduced fields.
- [ ] Omitted property policy.
- [ ] Validation status.
