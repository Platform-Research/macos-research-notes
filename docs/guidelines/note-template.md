# Note Template

Use this structure for `knowledge/**` notes.

```markdown
# Title

## Summary

One paragraph explaining what this note captures.

## Context

- macOS version:
- Artifact:
- Path:
- Tooling:

## Observations

- Directly observed facts.

## Inferences

- Reasoned conclusions based on observations.

## Unknowns

- Questions not answered yet.

## Related

- Related classes:
- Related kexts:
- Related frameworks:
- Related notes:

## Sources

- Local artifact:
- Command:
- External reference:
```
